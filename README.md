## 🧠 About This Repository

Welcome to my Deep Learning journey.

This repository contains hands-on implementations, experiments, and practical explorations of core and advanced deep learning concepts.

Unlike tutorial-style projects, this repo focuses on:

- Understanding algorithms from scratch  
- Implementing backpropagation manually  
- Exploring training dynamics  
- Debugging neural networks  
- Reproducing real-world architectures  
- Building strong conceptual foundations  

Every practical here is done with the goal of becoming a **deep learning engineer — not just a framework user.**
